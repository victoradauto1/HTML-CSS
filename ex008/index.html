Principais formatações

Negrito/ Destaque

Nesta frase usaremos este termo em negrito , usando a tag < b >

Nesta frase usaremos este termo em destaque , usando a tag < strong >

Itálico/ Ênfase

Nesta frase usaremos este termo em itálico usando a tag < i >

Nesta frase usaremos este termo um ênfase usando a tag < em >

Marcação de texto

Podemos marcar um texto usando a tag < mark >.

Texto grande e texto pequeno

Estamos criando um texto com letra grande e letra pequena, usando rspectivamente as tags < big > (deprecada) e < small > .

Texto Deletado

É o texto em que a parte deletada (não mais existente) é caracterizada com o efeito de riscado, para tal utilizaremos a tag < del >.

Texto Inserido

É um texto que fora acrescentado, por conseguinte, receberá um efeito de sublinhado, mas com efeitos semânticos para o site. Será feito utilizando a tag < ins >.

Texto sublinhado

Não é aconselhado efeito da tag < u > para se conseguir o efeito de sublinhado, uma vez que tal tag se preocupa apenas com a forma, e como já vimos, forma é atribuição de CSS. Tanto < u >, quanto < ins >, possuirão o mesmo efeito visual, porém este último carrega o aspecto semantico consigo.

Texto sobrescrito

Quando desejamos colocar parte do texto um patamar acima daquela mesma linha. X20+30. Utlizaremos para tal a tag <sup > .

Texto subrscrito

Quando desejamos coloca uma parte do texto num pequeno desnível. H2O. Utilizaremos a tag < sub >
